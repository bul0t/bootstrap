# Text
Работа с текстом: https://getbootstrap.com/docs/5.2/utilities/text/

## Text alignment

    text-start
    text-center
    text-end
    text-sm-start
    text-md-start
    text-lg-start
    text-xl-start

- `text-center text-md-start` - на маленьких экранах текст по центру, на больших слева.

## Reset color

    text-reset
